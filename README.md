# AI Clean Copy（AI 纯净复制）

> **面向 AI 内容复制场景的文本清洗工具。用于优化从 ChatGPT、Gemini 等页面复制后的文本质量，专注解决中英文空格异常、消灭网页暗藏字符残留等排版痛点。**

## 项目简介

在使用ChatGPT、Gemini等大语言模型工具生成内容后，用户通常需要将文本复制到Word、Markdown编辑器或其他文档处理环境中继续整理。

但直接从网页复制内容时，往往会带来一系列格式问题，例如：

- 中文与英文字母、数字之间混入不必要的冗余空格，极度影响阅读和后续排版（这在 AI 生成结果中尤为常见）；
- 网页中的隐藏字符、零宽 Unicode 字符或暗装的残留样式被一并带出；
- 段落、换行等基础格式在跨环境复制到 Word 等编辑器时表现极不稳定。

**AI Clean Copy** 主要用于处理上述问题。它针对主流 AI 平台的内容输出结构进行清洗与转换，帮助用户在复制文本时获得更规整、更适合二次编辑的结果。

目前已针对 **ChatGPT、Gemini、DeepSeek、Claude、Kimi** 等平台的常见内容结构进行了适配和优化。

## 功能特性

### 1. 文本清洗与排版优化

- 自动移除零宽字符等隐藏内容，减少复制后出现的异常空白或不可见字符；
- 优化中文与英文、数字混排时的空格问题，减轻复制后手动逐段修正的负担；
- 合并不必要的空行，保持段落结构相对整洁；
- 降低网页原始 DOM 结构和样式对粘贴结果的干扰。

### 2. 独立纯净复制模式

- **纯净文本模式**：剥离各种隐藏样式污染与乱码，提取极度干净的基础文本；
- **排版文本模式**：在控制冗余不可见代码的前提下，尽量保留标题、加粗、列表等有助于视觉阅读的常见层级。

### 3. 便捷高效流程

- 通过简单明确的组合键触发无感清洗复制；
- 彻底告别频繁手动粘贴后再逐句删空格、排查乱码空行的低效劳动；
- 深度赋能高频整理 AI 资料的写作、总结、翻译、报告撰写场景。

## 适用场景

本工具主要适用于以下使用情境：

- 将 ChatGPT、Gemini 等生成的繁杂中英混排内容丝滑、干净地复制并粘贴至 Word；
- 将冗余的 AI 网页返回内容纯净地归档到笔记软件、知识库或论文草稿中；
- 彻底避免因 AI 平台中文字与英文字母间距异常（常常表现为防不胜防的莫名空格）导致的反反复复人工清理。

其中，**中文与英文之间的多余空格问题**是最常见也最影响体验的场景之一。  
在直接复制 AI 页面内容时，这类空格通常会破坏中文阅读节奏，也会增加排版整理成本。AI Clean Copy 会对此类情况进行针对性处理，以提升复制结果的一致性和可用性。

## 安装方法

1. 下载或克隆本仓库到本地目录；
2. 打开 Chrome / Edge 浏览器扩展管理页面：`chrome://extensions/`；
3. 开启右上角的**开发者模式**；
4. 点击**加载已解压的扩展程序（Load unpacked）**；
5. 选择本项目所在文件夹完成安装。

## 使用说明

### 快速使用

1. 在网页中选中需要复制的文字、公式或代码内容；
2. 按下 `Ctrl + C`（macOS 下为 `Cmd + C`）；
3. 在设定时间窗口内按一次 `Space` 触发清洗复制；
4. 切换到 Word、Markdown 编辑器或其他目标软件中粘贴即可。

### 补充说明

- 也可以通过浏览器扩展图标打开插件面板，进行设置或手动触发复制；
- 若目标内容包含复杂公式或特殊网页结构，实际复制效果可能会因页面实现方式不同而有所差异。

## 可配置选项

通过点击浏览器扩展图标，可以进入设置面板。当前支持的选项包括：

- 智能紧缩处理中英文字母间异常的排版间距；
- 原样保留特殊独立段落的前端编排缩进；
- 强力清理各类 Unicode 零宽乱码字符；
- 折叠多余空行。

用户可根据自己的文档整理习惯进行调整。
