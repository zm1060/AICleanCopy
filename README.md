> **面向数字创作者的终极内容清洗工具。一键净化 AI 生成格式，完美对接 Word，告别排版地狱。**

## 为什么需要 AI Clean Copy？

在使用大语言模型工具时，我们经常需要将其生成的内容复制到 Word 或其他文档中。但传统的网页复制不仅会带入多余的 DOM 结构、黑底背景框、不可见字符，更会导致**所有数学公式（KaTeX/MathJax）在转换后变成破碎的乱码**。

**AI Clean Copy** 为解决此痛点而生：
完全兼容且针对 **ChatGPT、Gemini、DeepSeek、Claude、Kimi** 等主流 AI 平台的排版特征进行了调优。只需一个快捷键（`Ctrl+C → Space`），它将自动拦截并深度清洗剪贴板内容，为您送上格式正确、纯粹、公式完好的高质量文本结构。

---

## ✨ 核心亮点 (Features)

- 🧮 **极其强大的数学公式兼容**
  - 不仅支持传统的 `MathJax` 和 `KaTeX`，完美兼容 DeepSeek 的最新 `math-block` 和 `data-math` 直出模式！
  - 划选一段内容，即使跨越多个段落，数学公式也能原封不动转为优雅的 `$$...$$` 和 `$...$` LaTeX 代码段，轻松导入到任何 Markdown 或 Word 插件内。
- 🧹 **深度排版净化（告别“乱码”）**
  - 自动粉碎所有 U+200B 等**零宽特殊字符 / 幽灵空白**。
  - 自动修复中英文间距（智能去除汉字与英文间的突兀空格）。
  - 连接连续的碎化空行，确保你的段落紧凑干净。
- 📝 **双重复制引擎**
  - **【纯净文本模式】 (`Ctrl+C → Space`)**：最稳妥！丢弃多余样式，保留结构换行，彻底防污染。
  - **【清洁富文本模式】 (`Ctrl+V → Space`)**：完美保留标题层级、加粗、代码块以及表格，直接在 Word 中呈现结构。
- 🚀 **极致极客的交互体验**
  - 没有多余的快捷键冲突！选中文字，按下 `Ctrl+C` 触发复制，**0.5秒内**轻轻点击一下`空格键 (Space)`，即可触发自动清洗并吸入剪贴板！
- 💎 **高级感满满的现代 UI 配置页**
  - 采用全新的拟态毛玻璃（Glassmorphism）与赛博霓虹风格设计，好看就是战斗力！

## 📦 如何安装与使用

### 安装指南
1. 下载/克隆本仓库到本地文件夹。
2. 打开 Chrome / Edge 浏览器的扩展管理页面 `chrome://extensions/`。
3. 开启右上角的 **“开发者模式”**。
4. 点击 **“加载已解压的扩展程序” (Load unpacked)**，选择本文件夹即可！

### 极速使用教程
1. 用鼠标在网页中高亮选中你需要保存的代码、段落或公式。
2. 按下 `Ctrl + C`（苹果 Mac 为 `Cmd + C`）。
3. **松开双手，紧跟着敲击一下 `空格键`（Space）！** （屏幕上会弹出优美的成功提示角标 Toast）。
4. 切换到你的 Word 或者其他软件，执行粘贴。干净利落，大功告成！

_注：您也可以点击浏览器右上角拼图图标 🧩 打开插件进行设置或手动点击按钮进行复制。_
